What is Steganography

Steganography, As kids we were all fascinated while playing with ‘disappearing ink’. This ink was frequently used for passing secret messages from one person to another.

  • The sender used this ink to write secret messages on a piece of paper.
  • After writing, soon the ink would disappear.
  • This paper was then passed on to the receiver.
  • The receiver could read the message by moving a lighted candle over the paper.
  • Thus, the disappearing ink facilitated transmission of secret messages.

In digital systems this concept is called ‘Steganography’. Steganography is a technique of concealing messages or information within other non-secret text or data[google]. Using steganography, data can be hidden in images, videos or audio files.

The simplest medium to steganograph are images. There are various methods of performing Image Steganography:

  1. Least significant bit (LSB)
  2. Pixel value differencing (PVD)
  3. Edges based data embedding method (EBE)
  4. Random pixel embedding method (RPE)
  5. Mapping pixel to hidden data method
  6. Labelling or connectivity method
  7. Pixel intensity based method
  8. Texture based method
  9. Histogram shifting method

Lets discuss LSB based image steganography….

We know that images are made up of pixels. Each pixel stores a 24 bit RGB value as color code(8 bits per pixel; range of each color is 0-255). If the LSB of this color code is changed from 0 to 1 or vice versa, the color value changes by ± 1. This change in color value changes the shade of color but this change is not noticeable by the naked eye.(viewers can’t tell the difference between the original and stenographed image)

Steganographers exploit this phenomenon to hide secret data into images. They replace the LSB of pixel values by bits of data that is supposed to be hidden. This hidden data can be retrieved by analysing the LSB of pixel value of a steganographed image. Following example shows how the character ‘A’ is hidden in pixels.

Applications of Steganography

  1. Secret communication
  2. Digital watermarking
  3. Protection against piracy of media files.
